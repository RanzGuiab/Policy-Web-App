# Policy Atlas - Apps Script Showcase

Public portfolio version of a Google Apps Script knowledge portal.

## What this demonstrates
- Spreadsheet-driven navigation and metadata
- Google Docs embed preview loading
- Mixed media support (images, slides, audio, and video)
- Client-side search, modal details, and responsive navigation

## Notes
- Backend method contracts are preserved so existing Apps Script flows still work.
- Use non-sensitive sample content before publishing.
